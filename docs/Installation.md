# Installation

## Prerequisites

- **Node.js** >= 14.0.0
- **npm** >= 6.0.0

## Setup

```bash
git clone https://github.com/turkcode-ai/next-shield.git
cd next-shield
npm install
```

## Verify Installation

```bash
node index.js --version
node index.js --help
```

## Troubleshooting

If you encounter issues during setup:

1. Verify your Node.js version: `node --version`
2. Clear npm cache: `npm cache clean --force`
3. Delete `node_modules` and reinstall: `rm -rf node_modules && npm install`

Still having problems? [Open an issue](https://github.com/turkcode-ai/next-shield/issues).
